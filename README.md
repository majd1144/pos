This project is a Point of Sale (POS) system implemented as part of my internship training.  
The system is designed to manage employees, customers, products, orders, payments, and discounts using a relational database in SQL Server.



*Project Features:
- Employee management with roles and active status  
- Customer management with personal details and activity status  
- Product and category management with tax and total calculations  
- Order creation and management including order details  
- Payment recording and automatic order status updates  
- Discount management with detailed rules and permissions  
- Triggers to maintain data consistency:  
  - Auto-fill discount amounts  
  - Update order totals, net totals, and discount amounts  
  - Sync product info with order details  
  - Update order status after payments




*Tech Stack:
- Database:SQL Server (T-SQL)  
- Key Concepts: Triggers, Foreign Keys, Computed Columns, Constraints, CRUD operations  
- Role:Database design, triggers implementation, and data manipulation


*Sample Queries:
- Retrieve top-selling products  
- Calculate net totals and discount amounts  
- Find orders with multiple products  
- Filter orders by date or discount usage
